Version History

Version 4.0

Released: January 4, 2025

  • Added a new automated cash flow summary sheet.
  • Improved accuracy of forecasting formulas for income and expenses.
  • Updated instructions for better user onboarding experience.
  • Fixed minor bugs in conditional formatting rules.

Version 3.0

Released: December 1, 2024

  • Introduced support for customizable categories in the budget sheet.
  • Added compatibility with older Excel versions.
  • Enhanced visualization with a new bar chart for monthly savings.

Version 2.0

Released: October 15, 2024

  • Improved performance for larger datasets.
  • Refined the income vs. expenses comparison tool.
  • Minor UI adjustments to align sheets for better readability.

Version 1.0

Released: September 1, 2024

  • Initial release with core budget forecasting functionality.
  • Included income, expense, and savings tracking features.
  • Basic pie chart visualization for spending breakdown.
